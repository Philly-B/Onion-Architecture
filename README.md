# Onion-Architecture

This is a test project which is following the concept of the onion architecture. It's goal is to be as capsulated as possible, so that external libraries/frameworks can be exchanges as easily as possible. The overall concept is described here](https://www.thinktocode.com/2018/08/16/onion-architecture/).
